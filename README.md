# Editable Signature Template

This workspace now includes an editable HTML signature template based on the signature PNG.

## Files added

- `signature-template.html` — live editor and preview for a signature card
- `signature-template.css` — styling for the card and editor

## How to use

1. Open `signature-template.html` in your browser.
2. Edit the fields on the left.
3. The preview updates immediately.

## Customization

- Replace the default logo box with your own image by editing the HTML in `signature-template.html`.
- Change the default text values in the form `input` elements.
- Use the preview to recreate this card design for other people.
